二.过程

  1. 子序列的划分：将待排数组分为gap个子序列(距离为gap的元素放在同一自序列中)
  2. 排序：对每个子序列分别实行直接插入排序
  3. 缩小gap，直到gap=1,重复子序列的划分个排序操作

三.例题

四.时间复杂度

用实例分析：
序列待排序的元素n=200，取步数d=40,13,4,1

  1. d=40时，划分成40个子序列
    $ { S{0},S{40},S{80},S{120},S{160} }$… $ { S{39},S{79},S{119},S{159},S{199} } $
    对40个序列使用插入排序（ $ N_{1} $=15）
  2. d=13时，划分成13个子序列
    $ { S{0},S{13},S{26},…,S{194} } $ … $ { S{12},S{25},S{38},…,S{193} } $
    对13个序列使用插入排序($ N_{2} $=15)
  3. d=4，划分成4个子序列
    $ { S{0},S{4},…,S{196} } $… $ { S{3},S{7},…,S{199} } $
    对4个子序列使用插入排序($ N_{3} $=50)
  4. d=1，划分成1个子序列
    对1个子序列使用直接插入排序($ N_{4} $=200)

由直接插入排序的时间复杂度O(N^2)
的总复杂度为：
$$ 40N_{1}^{2}+13N{2}^{2}+4*N{3}^{2}+1N_{3}^{2}=53925 $$
这结果比直接1（$ N^{2} $）=40000大，但是经过第一次排序，后续子序列成为近似排序了，因此实际次数函数接近于n,所以实际KCN是：

$$ 40N_{1}^{2}+13N{2}+4*N{3}+1*N_{4}=1595 $$

所以时间复杂度为：

$$ O(N^{1.5}) $$
